Sneak Lite is a clean, minimal and responsive SuevaFree child theme.

== Description ==

Sneak Lite is a clean, minimal and responsive SuevaFree child theme, based on Bootstrap framework, with a new menu and buttons style and a different layout for the elements like sidebar and with a new fonts.

Created by ThemeinProgress, https://www.themeinprogress.com

Sneak Lite WordPress Theme is a child theme of SuevaFree Theme, Copyright 2023 ThemeinProgress.
Sneak Lite is distributed under the terms of the GNU GPLv3.
