== Description ==

Philia is a Full Site Editing (FSE) theme perfectly crafted for your next blog website. It is a simple, clean, minimal theme that can be used for blogs, newspapers, personal sites, small businesses, and many more. You can use the Philia theme to create your next website without any coding.
